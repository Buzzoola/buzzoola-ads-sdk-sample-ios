# Buzzoola Sample App

Buzzoola Sample App - это iOS-приложение на SwiftUI, в котором представлен пример интеграции мобильного рекламного SDK Buzzoola. В приложении показано, как быстро и правильно настроить показ рекламных форматов, используя наш SDK.

## Документация

Полная документация по интеграции и настройке SDK доступна по ссылке: 
🔗 [https://docs.buzzoola.com/publishers/in-app_sdk_integration/inapp_sdk_ios]

## Требования
Минимальная версия iOS: 13.0
